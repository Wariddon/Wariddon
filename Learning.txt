Dev Ops . : Dynamic Deploy System (per day) 
	- ทุกคนต้องมีความรู้เท่ากัน และ มี KPI ตรงกัน ( มุมมองเหมือนกัน )
	- Dev Ops ที่ดีคือคนที่มี communication ที่ดี

Agile . :     
	- คน และ ปฏิสัมพันธ์ของผู้คน ให้ความสำคัญกับผู้คนมากที่สุด ( communication )
	- ให้ความสำคัญกับผลงานให้สำเร็จและออกมาใช้ได้จริงมากกว่า documentation ก่อนจะใช้ เพื่อต้องการ feed back ให้เร็วและบ่อยที่สุด
	- การพูดคนกันระหว่างแต่ละทีม ช่วยเหลือกัน กันบ่อยๆ เมื่อทำตาม plan เสร็จโยนทิ้งไปและทำ plan ใหม่เลย
 โทษ : Agile ไม่ได้ทำงานได้เร็วขึ้น ,
